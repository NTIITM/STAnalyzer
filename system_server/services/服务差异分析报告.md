## 二、核心算法差异

### 2.1 SpaOTsc Service

**算法原理：**
- 基于**最优传输(Optimal Transport, OT)**理论
- 使用**质心距离**计算空间核函数
- 结合配体-受体表达量和空间距离计算通信分数

**核心公式：**
```
score = ligand_expr × receptor_expr × kernel(distance, bandwidth)
kernel = exp(-distance / bandwidth)
```

**关键代码实现：**
```373:414:system_server/services/spaotsc-service/main.py
def _compute_scores(
    group_means: pd.DataFrame,
    dist_df: pd.DataFrame,
    lr_pairs: List[Dict[str, str]],
    bandwidth: float,
    min_expr: float,
) -> pd.DataFrame:
    rows: List[Dict[str, Any]] = []
    groups = group_means.index.tolist()
    for entry in lr_pairs:
        ligand_vec = group_means[entry["ligand"]]
        receptor_vec = group_means[entry["receptor"]]
        for sender in groups:
            lig_expr = float(ligand_vec.loc[sender])
            if lig_expr < min_expr:
                continue
            for receiver in groups:
                rec_expr = float(receptor_vec.loc[receiver])
                if rec_expr < min_expr:
                    continue
                distance = float(dist_df.loc[sender, receiver])
                kernel = float(np.exp(-distance / max(bandwidth, 1e-6)))
                score = lig_expr * rec_expr * kernel
                if score <= 0:
                    continue
                rows.append(
                    {
                        "sender": sender,
                        "receiver": receiver,
                        "ligand": entry["ligand"],
                        "receptor": entry["receptor"],
                        "distance": distance,
                        "transport_kernel": kernel,
                        "score": score,
                    }
                )
    df = pd.DataFrame(rows)
    if df.empty:
        return df
    df["normalized_score"] = df["score"] / (df["score"].sum() or 1.0)
    return df.sort_values(by="score", ascending=False)
```

**特点：**
- ✅ 基于**质心距离**，适合分析**聚类级别**的空间通信
- ✅ 使用**指数衰减核函数**，距离越近权重越大
- ✅ 输出包含**距离、核值、分数**等详细信息
- ❌ 不考虑统计显著性检验

### 2.2 MISTY Service

**算法原理：**
- **多视角建模**：将空间影响分解为三个视角
  - **Intrinsic（内在）**：目标基因自身表达
  - **Juxtacrine（近邻）**：紧邻空间邻居的影响
  - **Paracrine（旁分泌）**：远距离配体信号的影响

**核心公式：**
```
prediction = w_intrinsic × intrinsic + w_juxtacrine × juxtacrine + w_paracrine × paracrine
weights = normalized(|correlations|)
```

**关键代码实现：**
```210:258:system_server/services/misty-service/main.py
def _compute_contributions(
    expr_df: pd.DataFrame,
    neighbor_df: pd.DataFrame,
    target_genes: List[str],
    ligand_set: List[str],
) -> Tuple[pd.DataFrame, pd.DataFrame]:
    results: List[Dict[str, Any]] = []
    predictions_rows: List[Dict[str, Any]] = []
    ligand_cols = [gene for gene in ligand_set if gene in expr_df.columns]
    ligand_neighbor_mean = neighbor_df[ligand_cols].mean(axis=1) if ligand_cols else pd.Series(0, index=expr_df.index)
    intrinsic_feature = expr_df.mean(axis=1)
    for gene in target_genes:
        if gene not in expr_df.columns:
            continue
        target_expr = expr_df[gene]
        juxta_feature = neighbor_df[gene]
        para_feature = ligand_neighbor_mean
        intrinsic_corr = float(np.corrcoef(target_expr, intrinsic_feature)[0, 1])
        juxta_corr = float(np.corrcoef(target_expr, juxta_feature)[0, 1])
        para_corr = float(np.corrcoef(target_expr, para_feature)[0, 1]) if not para_feature.equals(0) else 0.0
        corr_values = np.array([abs(intrinsic_corr), abs(juxta_corr), abs(para_corr)])
        weights = corr_values / (corr_values.sum() or 1.0)
        norm_intrinsic = (intrinsic_feature - intrinsic_feature.mean()) / (intrinsic_feature.std() or 1.0)
        norm_juxta = (juxta_feature - juxta_feature.mean()) / (juxta_feature.std() or 1.0)
        norm_para = (para_feature - para_feature.mean()) / (para_feature.std() or 1.0)
        prediction = weights[0] * norm_intrinsic + weights[1] * norm_juxta + weights[2] * norm_para
        predictions_rows.extend(
            [
                {
                    "cell_id": idx,
                    "target_gene": gene,
                    "predicted": float(pred),
                    "actual": float(target_expr.loc[idx]),
                }
                for idx, pred in zip(expr_df.index, prediction)
            ]
        )
        results.append(
            {
                "target_gene": gene,
                "intrinsic": weights[0],
                "juxtacrine": weights[1],
                "paracrine": weights[2],
                "intrinsic_corr": intrinsic_corr,
                "juxtacrine_corr": juxta_corr,
                "paracrine_corr": para_corr,
            }
        )
    return pd.DataFrame(results), pd.DataFrame(predictions_rows)
```

**特点：**
- ✅ **细胞级别**分析，不需要预先聚类
- ✅ 量化**三种空间影响**的相对贡献
- ✅ 基于**相关性**自动分配权重
- ❌ 需要指定**目标基因**或自动选择
- ❌ 不直接输出配体-受体对

### 2.3 Giotto-CPDB Service

**算法原理：**
- 结合**Giotto空间邻域**和**CellPhoneDB统计方法**
- 使用**置换检验**计算统计显著性
- 输出**z-score**和**p-value**

**核心公式：**
```
observed_score = ligand_expr × receptor_expr × spatial_weight
z-score = (observed - mean(null)) / std(null)
p-value = permutation_test(observed, null_distribution)
```

**关键代码实现：**
```232:268:system_server/services/giotto-cpdb-service/main.py
def _score_observed(
    group_means: pd.DataFrame,
    dist_df: Optional[pd.DataFrame],
    lr_pairs: List[Dict[str, str]],
    min_expr: float,
    spatial_radius: float,
) -> pd.DataFrame:
    rows: List[Dict[str, Any]] = []
    groups = group_means.index.tolist()
    for entry in lr_pairs:
        ligand_vec = group_means[entry["ligand"]]
        receptor_vec = group_means[entry["receptor"]]
        for sender in groups:
            lig_expr = float(ligand_vec.loc[sender])
            if lig_expr < min_expr:
                continue
            for receiver in groups:
                rec_expr = float(receptor_vec.loc[receiver])
                if rec_expr < min_expr:
                    continue
                spatial_weight = 1.0
                if dist_df is not None:
                    distance = float(dist_df.loc[sender, receiver])
                    spatial_weight = float(np.exp(-distance / max(spatial_radius, 1e-6)))
                score = lig_expr * rec_expr * spatial_weight
                rows.append(
                    {
                        "sender": sender,
                        "receiver": receiver,
                        "ligand": entry["ligand"],
                        "receptor": entry["receptor"],
                        "expr_score": lig_expr * rec_expr,
                        "spatial_weight": spatial_weight,
                        "score": score,
                    }
                )
    return pd.DataFrame(rows)
```

**置换检验实现：**
```271:298:system_server/services/giotto-cpdb-service/main.py
def _permutation_scores(
    expr_df: pd.DataFrame,
    groupby: str,
    lr_pairs: List[Dict[str, str]],
    spatial_scores: Dict[Tuple[str, str], float],
    groups: List[str],
    n_permutations: int,
    seed: int,
) -> DefaultDict[Tuple[str, str, str, str], List[float]]:
    rng = np.random.default_rng(seed)
    rows: DefaultDict[Tuple[str, str, str, str], List[float]] = defaultdict(list)
    labels = expr_df[groupby].values.astype(str)
    expr_only = expr_df.drop(columns=[groupby]).copy()
    for _ in range(n_permutations):
        shuffled = rng.permutation(labels)
        expr_only[groupby] = shuffled
        perm_means = expr_only.groupby(groupby).mean().reindex(groups).fillna(0.0)
        for entry in lr_pairs:
            ligand_vec = perm_means[entry["ligand"]]
            receptor_vec = perm_means[entry["receptor"]]
            for sender in groups:
                for receiver in groups:
                    expr_score = float(ligand_vec.loc[sender] * receptor_vec.loc[receiver])
                    if expr_score <= 0:
                        continue
                    spatial_weight = spatial_scores.get((sender, receiver), 1.0)
                    rows[(entry["ligand"], entry["receptor"], sender, receiver)].append(expr_score * spatial_weight)
    return rows
```

**特点：**
- ✅ **统计显著性检验**（置换检验）
- ✅ 输出**z-score**和**p-value**
- ✅ 基于**聚类级别**分析
- ✅ 可过滤**显著互作**
- ❌ 计算时间较长（需要多次置换）

### 2.4 NicheNet Service（已归档）

**算法原理：**
- 基于**配体-靶基因先验网络**
- 结合**接收群体差异表达**
- 计算**配体活性评分**

**核心公式：**
```
ligand_activity = max(ligand_expression)
target_delta = mean(receiver_expr - others_expr)
combined_score = ligand_activity × max(target_delta, 0)
```

**特点：**
- ✅ 使用**先验知识网络**
- ✅ 关注**接收群体的靶基因响应**
- ❌ 主要用于**单细胞数据**（非空间）
- ❌ 已归档，不在当前服务目录

## 七、结果解读建议

### 7.1 SpaOTsc结果解读
- **score**: 通信强度（越高越强）
- **distance**: 质心距离（越小越近）
- **transport_kernel**: 空间核值（1.0表示同一聚类）
- **建议**: 关注高分边，结合距离判断空间相关性

### 7.2 MISTY结果解读
- **intrinsic**: 内在因素贡献（目标基因自身）
- **juxtacrine**: 近邻贡献（紧邻细胞影响）
- **paracrine**: 旁分泌贡献（配体信号影响）
- **建议**: 如果paracrine贡献高，说明受空间配体信号影响大

### 7.3 Giotto-CPDB结果解读
- **zscore**: 标准化分数（>2通常认为显著）
- **pvalue**: 统计显著性（<0.05通常认为显著）
- **建议**: 优先关注`giotto_cpdb_significant.csv`中的显著互作

## 八、总结

### 核心差异总结

1. **分析粒度**:
   - SpaOTsc & Giotto-CPDB: **聚类级别**
   - MISTY: **细胞级别**

2. **统计方法**:
   - SpaOTsc: **无统计检验**
   - MISTY: **相关性分析**
   - Giotto-CPDB: **置换检验**

3. **输出重点**:
   - SpaOTsc: **通信分数和距离**
   - MISTY: **空间影响类型**
   - Giotto-CPDB: **统计显著性**

4. **计算复杂度**:
   - SpaOTsc: **低**（O(n²)聚类对）
   - MISTY: **中**（O(n)细胞，需邻居计算）
   - Giotto-CPDB: **高**（需要多次置换）

### 选择建议

- **快速探索**: 使用 **SpaOTsc**
- **深入机制**: 使用 **MISTY**
- **统计验证**: 使用 **Giotto-CPDB**
- **综合使用**: 先用SpaOTsc筛选，再用Giotto-CPDB验证显著性
