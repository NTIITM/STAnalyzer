Spatial transcriptomics has emerged as a revolutionary technology that enables the simultaneous measurement of gene expression and spatial location information, providing unprecedented insights into tissue architecture and cellular interactions. The integration of spatial context with transcriptomic data has significantly advanced our understanding of complex biological processes, disease mechanisms, and developmental patterns. However, the lack of intelligent analytical platforms that seamlessly integrate knowledge retrieval, automated analysis, and code generation remains a significant hindrance for researchers to efficiently analyze spatial transcriptomic data and extract meaningful biological insights.

The elucidation of spatial gene expression patterns and their functional implications drives the development of analytical platforms and tools, such as Seurat (Hao et al., 2021, Cell), Scanpy (Wolf et al., 2018, Genome Biology), Squidpy (Palla et al., 2022, Nature Methods), and Giotto (Dries et al., 2021, Nature Biotechnology). Although these platforms have made significant advancements in spatial transcriptomics analysis, they primarily focus on computational workflows and lack intelligent task planning capabilities. Current tools require researchers to manually design analysis pipelines, write custom code, and integrate knowledge from multiple sources, leading to fragmented workflows and time-consuming processes. Furthermore, existing platforms do not systematically integrate private knowledge bases with public literature retrieval, nor do they provide automated code generation based on natural language queries. These limitations significantly impede the efficiency of spatial transcriptomics research and limit the accessibility of advanced analytical capabilities to non-computational biologists.

Therefore, we present textMSA, which features the first-of-its-kind intelligent analysis system that aims to revolutionize spatial transcriptomics research through AI-powered task planning, multi-source knowledge integration, and automated code generation. textMSA leverages LangGraph-based agent architecture to automatically decompose complex research questions into executable tasks, intelligently routes between knowledge retrieval and data analysis workflows, and generates comprehensive answers by integrating results from multiple sources. The system seamlessly integrates private knowledge bases (through vector-based semantic search) with public literature retrieval from multiple sources including PubMed, arXiv, Semantic Scholar, CrossRef, Google Scholar, and web search, providing researchers with comprehensive knowledge support. To support intelligent data analysis, textMSA incorporates specialized services for spatial transcriptomics data processing, including H5AD file reading, spatial spot visualization, gene expression analysis, and multi-modal data integration. Advancing beyond static analysis pipelines, textMSA also provides innovative automated code generation capabilities that translate natural language queries into executable analysis code, supporting multiple programming languages (Python, R, Julia, Bash) and conda environment management. The system is built on a modular, extensible architecture using FastAPI and MongoDB, supporting multi-user, multi-project collaborative analysis with complete data isolation and persistent storage. To maintain scientific currency and adaptability, textMSA features a pluggable service architecture that allows easy integration of new analysis tools and visualization types, ensuring the platform can evolve with rapidly advancing spatial transcriptomics technologies.
